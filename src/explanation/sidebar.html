Sidebar Menu met React Router – Uitgebreide uitleg

1. Imports van React Router

import {
  BrowserRouter as Router,
  Routes,
  Route,
  NavLink,
  useLocation
} from "react-router-dom";

Uitleg per onderdeel:

  • BrowserRouter (Router):
    • Het container-element dat routing in React mogelijk maakt.
    • Alles wat navigatie bevat, moet binnen deze component staan.
    • In de sidebar app zorgt Router ervoor dat de URL verandert bij klik op NavLink.
  • Routes:
    • Bevat alle mogelijke routes (Route elementen).
    • React Router kijkt naar de huidige URL en rendert het juiste component.
    • In combinatie met NavLink maakt het navigatie mogelijk.
  • Route:
    • Definieert één route: bijvoorbeeld <Route path="/home" element=<Home /> />.
    • Elk submenu-item kan naar een Route verwijzen via NavLink.
  • NavLink:
    • Wordt gebruikt voor links die navigeren naar een route.
    • Kan automatisch detecteren of de route actief is (isActive) en een speciale klasse toevoegen (active-link).
    • In jouw sidebar wordt NavLink gebruikt voor submenu-items.
  • useLocation:
    • Hook die informatie geeft over de huidige URL.
    • Wordt gebruikt om bijvoorbeeld te checken welk menu-item actief is of extra logica uit te voeren wanneer de route verandert.

2. Sidebar-menu structuur

<nav className={`sidebar ${open ? "open" : ""}`}>
  <ul className="menu">
    {menu.map((item) => (
      <li key={item.path}>
        <div className="menu-label" 
             onClick={() => item.children && toggle(item.label)}>
          {item.children && (
            <span className={`arrow ${expanded[item.label] ? "down" : "right"}`}>▶</span>
          )}
          {item.label}
        </div>

        {item.children && expanded[item.label] && (
          <ul className="submenu">
            {item.children.map((child) => (
              <li key={child.path}>
                <NavLink
                  to={child.path}
                  onClick={() => isMobile && setOpen(false)}
                  className={({ isActive }) => (isActive ? "active-link" : "")}
                >
                  {child.label}
                </NavLink>
              </li>
            ))}
          </ul>
        )}
      </li>
    ))}
  </ul>
</nav>

3. Uitleg van de elementen

  • <nav>: Container voor de sidebar, klasse open bepaalt of hij zichtbaar is.
  • <ul class="menu">: Lijst van hoofdmenu-items.
  • <li>: Elk menu-item.
  • <div class="menu-label">:
    • Neutraal klikbaar element om submenu uit te klappen.
    • Waarom geen NavLink? Omdat hoofdmenu-items soms alleen een submenu openen, geen route navigeren.
  • <span class="arrow">: Pijltje dat visueel laat zien of submenu open of dicht is.
  • <ul class="submenu">: Submenu-items, alleen zichtbaar als expanded[item.label] true is.
  • <NavLink>:
    • Navigeert naar een route (to={child.path}).
    • Geeft automatisch active-link wanneer de route actief is.
    • Verbindt de import van React Router direct met de submenu-links.

4. Uitleg van functies en state

  • toggle(label):
    • Opent of sluit het submenu voor een menu-item.
    • Voorbeeld: 
      function toggle(label) {
  setExpanded(prev => ({
    ...prev,
    [label]: !prev[label]  // Wisselt de boolean van dit item
  }));
}
    • prev staat voor de vorige state van expanded, zodat andere submenu-items niet beïnvloed worden.
  • expanded[item.label]: boolean die aangeeft of een submenu open is.
  • setOpen(false): sluit sidebar op mobiel na klik op submenu-item.
  • isActive in NavLink: markeert actieve link visueel.
  • useLocation: geeft informatie over huidige URL, bijvoorbeeld voor logica van actief menu-item.

5. Connectie tussen React Router import en de sidebar

  • BrowserRouter / Router: alle NavLink elementen werken alleen binnen deze container. Zonder Router zou navigatie niet functioneren.
  • Routes en Route: bepalen welke component getoond wordt bij een URL, bijvoorbeeld wanneer een submenu-link wordt aangeklikt.
  • NavLink: maakt de connectie tussen submenu-items en de Route. Zorgt dat klik op link de URL verandert en juiste component rendert.
  • useLocation: kan gebruikt worden om te checken welke Route actief is en zo bijvoorbeeld submenu automatisch open te zetten of actieve link te highlighten.

6. Samengevat

Elementen en functies:

  • div voor hoofdmenu-items: opent submenu, geen navigatie
  • NavLink voor submenu-items: navigeert + markeert actief
  • toggle(label), expanded, prev, setOpen(false): beheren submenu state en mobiele sidebar
  • React Router import (Router, Routes, Route, NavLink, useLocation) verbindt menu met navigatie en actieve status van links
